## [1.0.0] — 2026-06-15

### Added
- Initial release of the Dungeon Gadgets landing page
- Next.js 16 (App Router) + Tailwind CSS v4 scaffold
- Dark fantasy aesthetic — Cinzel display font, EB Garamond body, parchment/gold colour palette
- Animated canvas particle system with per-tool accent colours (emerald, amber, rose, violet)
- Breathing radial gradient background overlay with noise grain texture
- Full-page single-viewport layout: header, hero, tool list, footer — no scrolling required
- Hero section featuring the Dungeon Gadgets chest illustration with purple/gold drop-shadow glow
- Header with stone-text wordmark logo and nav links
- Tool list with three entries — Lineage, Dungeon Tracker, Token Forge — each with coloured accent dot, hover left-bar reveal, and row wash effect
- Footer with About, GitHub, Report an Issue links; MIT license badge; copyright
- Gold shimmer gradient lines on header and footer borders
- Responsive layout — stacks vertically on mobile
- `index.html` static fallback (CDN Tailwind) retained for reference
